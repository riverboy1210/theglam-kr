## 🚨 추가가 필요한 항목 (11개)

### 11. 커뮤니티 / 게시판 시스템 설계 ⭐ 핵심 누락

> 사이트의 핵심 목표 중 하나인 **"서로 이야기를 나눌 수 있는"** 기능이 기획서에 없습니다.

| 항목 | 내용 |
|------|------|
| **필요성** | 바비톡의 핵심 경쟁력은 350만 건 게시글 — 즉 **커뮤니티**임. 단순 댓글이 아닌 독립 게시판이 필요 |
| **게시판 유형** | ① **자유 게시판** (고민·질문·잡담) ② **시술별 게시판** (눈/코/지방흡입 등 분류) ③ **Before/After 인증 게시판** ④ **병원 후기 게시판** ⑤ **이벤트 정보 공유** |
| **핵심 기능** | 글 작성(사진 첨부), 댓글/대댓글, 좋아요, 스크랩, 인기글 자동 선별, 신고/차단 |
| **익명성 정책** | 민감한 주제이므로 **닉네임 기반 익명** 운영 (카카오/네이버 로그인 후 닉네임 설정) |
| **SEO 연계** | 게시판 글도 검색엔진에 인덱싱 → 롱테일 키워드 자동 확보 (예: "눈 재수술 3개월 후기") |
| **CTA 연계** | 게시판 글 사이드바에 관련 병원 추천 카드·상담신청 CTA 자동 삽입 |
| **우선순위** | **P0 (필수)** — Phase 2에 통합 권장 |
| **기술 고려** | Webflow 자체로는 게시판 구현 불가 → 외부 서비스(Discourse, Circle 등) 임베드 또는 커스텀 개발 필요 |

### 12. 실시간 채팅 / 1:1 상담 기능

| 항목 | 내용 |
|------|------|
| **필요성** | 상담 폼만으로는 즉시 응답이 불가. 강남언니는 앱 내 실시간 상담을 지원 |
| **구현 제안** | ① **카카오톡 채널 연동** (1차, 가장 빠름) ② **실시간 채팅 위젯** (채널톡, Tawk.to 등) ③ **AI 챗봇** (자주 묻는 시술 질문 자동 응답 → 상담 전환) |
| **CTA 위치** | 페이지 우하단 플로팅 채팅 버튼, 기사 본문 내 "지금 바로 상담하기" |

### 13. 검색 기능 고도화

| 항목 | 내용 |
|------|------|
| **현재 문제** | 사이트 내부 검색이 없거나 기본 수준. 콘텐츠가 늘어날수록 필수 |
| **구현 제안** | ① **통합 검색** (기사 + 후기 + 병원 + 커뮤니티 게시글) ② **자동완성** (인기 검색어 기반) ③ **필터** (시술 종류, 지역, 가격대, 평점) ④ 바비톡의 **AI 검색(RAG)** 벤치마크로 장기 목표 설정 |
| **SEO 효과** | 내부 검색 데이터 → 인기 키워드 발굴 → 신규 콘텐츠 기획에 활용 |
| **우선순위** | **P1** — Phase 2에 기본 검색, Phase 4에 AI 검색 |

### 14. 신뢰도 및 브랜딩 전략

| 항목 | 내용 |
|------|------|
| **필요성** | 의료 정보 콘텐츠의 핵심은 **신뢰**. 기획서에 신뢰도 확보 전략이 빠져 있음 |
| **구현 제안** | ① **전문의 자문단** 표시 (실제 의사의 감수 명시) ② **에디터 프로필 강화** (현재 4명 → 전문성 근거 추가) ③ **의료 정보 면책 고지** (법적 필수) ④ **출처 표기 정책** (논문, 공식 가이드라인 인용 시) ⑤ **브랜드 가이드라인** (톤 앤 매너, 로고, 컬러 규정) |
| **특이사항** | Google E-E-A-T(경험·전문성·권위·신뢰) 기준 충족이 SEO 순위에 직결됨 |
| **우선순위** | **P0 (필수)** — 모든 Phase에 걸쳐 적용 |

## 🔄 기존 로드맵 수정 제안

기존 4 Phase 로드맵에 추가 항목을 통합하면:

### Phase 0 (사전 준비, 2주)
- ⚖️ 법적 검토 (의료광고법, 개인정보보호법)
- 🎨 브랜드 가이드라인 확정
- 📋 이용약관·개인정보 처리방침 작성
- 🌐 커스텀 도메인(theglam.kr) 연결

### Phase 1 (MVP, 1~2개월) — 기존 + 추가
- 기존 항목 유지
- ➕ SNS 공유 버튼, 네이버 검색 등록
- ➕ 카카오톡 채널 연동 (1:1 상담)
- ➕ Microsoft Clarity 설치 (무료 히트맵)
- ➕ E-E-A-T 기반 에디터 프로필 강화
- ➕ 의료 정보 면책 고지 추가

### Phase 2 (3~4개월) — 기존 + 추가
- 기존 항목 유지
- ➕ **커뮤니티 게시판** (자유·시술별·후기) ⭐
- ➕ 사이트 내부 통합 검색
- ➕ 이메일 뉴스레터 시스템
- ➕ 콘텐츠 관리 워크플로우 수립

### Phase 3 (5~6개월) — 기존 + 추가
- 기존 항목 유지
- ➕ 커뮤니티 관리 도구 (신고·필터링)
- ➕ 카카오 알림톡 연동
- ➕ A/B 테스트 체계
- ➕ 게시판 콘텐츠 SEO 인덱싱

### Phase 4 (7개월~) — 기존 + 추가
- 기존 항목 유지
- ➕ AI 검색 (RAG 기반)
- ➕ 온보딩 퀴즈·퍼소나 경험
- ➕ 비활성 유저 재참여 자동화

## 💡 핵심 한 줄 요약

> 기존 기획서는 **"콘텐츠 → 전환"** 중심이고, 부족한 것은 **"소통 → 체류 → 재방문"** 쪽입니다.
> **커뮤니티 게시판**이 가장 큰 빈 자리이며, 이것이 바비톡·강남언니와 경쟁할 핵심 무기입니다.
